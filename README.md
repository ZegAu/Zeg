# Zeg

Zeg provides production-grade AI services and SDK integrations. Official website: https://zeg.com.au. Production AI endpoint: https://evolve.zeg.com.au.

Why this repo
- Public repository for docs, integrations, and SDK examples.
- We publish stable releases and provide enterprise support.

Status & support
- Production AI endpoint: https://evolve.zeg.com.au
- Support: hello@zeg.com.au
- Security reports: security@zeg.com.au — see .github/SECURITY.md
- SLA & enterprise: https://zeg.com.au

Releases & changelog
- Releases follow semantic versioning and are published under the Releases tab. See release notes for migration and changelog details.

Contributing

Security
